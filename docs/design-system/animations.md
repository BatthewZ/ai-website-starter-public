# Animations

The system provides built-in animation keyframes:

- `animate-fade-in`
- `animate-fade-down`
- `animate-fade-up`
- `animate-fade-right`
- `animate-fade-left`
- `animate-slide-in-right`
- `animate-slide-out-right`

All keyframes use `--MOTION-*` tokens for durations and easing, so they can be re-themed globally.
