# Media Tokens

**File:** `src/web/style/tokens/media.css`

Sizing and behavior tokens for media-rich components (cards, carousels, galleries).

| CSS Variable               | Default Value     | Purpose                                    |
| -------------------------- | ----------------- | ------------------------------------------ |
| `--MEDIA-ASPECT-POSTER`    | `2 / 3`           | Portrait poster ratio                      |
| `--MEDIA-ASPECT-WIDE`      | `16 / 9`          | Widescreen ratio                           |
| `--MEDIA-ASPECT-SQUARE`    | `1 / 1`           | Square ratio                               |
| `--MEDIA-CARD-HOVER-SCALE` | `1.05`            | Card zoom on hover                         |
| `--MEDIA-CARD-HOVER-LIFT`  | `-4px`            | Card Y-translate on hover                  |
| `--MEDIA-CAROUSEL-PEEK`    | `48px`            | Visible portion of next/prev carousel item |
| `--MEDIA-CAROUSEL-GAP`     | `var(--R-SIZE-5)` | Gap between carousel items                 |
