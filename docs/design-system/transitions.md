# Transitions

**File:** `src/web/style/tokens/transitions.css`

Simple interaction transition durations for hover states and micro-interactions.

| CSS Variable       | Default Value | Tailwind Class    | Usage                       |
| ------------------ | ------------- | ----------------- | --------------------------- |
| `--DURATION-FAST`  | `100ms`       | `duration-fast`   | Hover states, color changes |
| `--DURATION-NORMAL`| `200ms`       | `duration-normal` | Transforms, layout shifts   |
| `--DURATION-SLOW`  | `300ms`       | `duration-slow`   | Entrance animations, modals |
